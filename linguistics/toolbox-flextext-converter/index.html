Toolbox - Flextext converter

For linguistic transcription files.

This takes Toolbox's xml export format, and converts it into flextext xml format. It is an XSL script, which therefore runs in various XSL processors such as Saxon.

You can download the XSL script here - it's bundled up with tiny samples of the input Toolbox format and the output Flextext format.

Toolbox xml export

Note that exporting xml from Toolbox requires a non-obvious move. From File > Export, you need to click the ADD option, that will allow you to add xml as an export format option.
